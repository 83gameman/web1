    01. WWW(World Wide Web) 인터넷에 연결된 컴퓨터들을 통해 정보를 공유할 수 
    있는 전 세계적인 정보 공간을 말한다. 줄여서 웹(Web)
	
    - OSI(Open System Interconnection) 7 Layer Model
           OSI 7 계층 모형,프로토콜을 기능별로 나눈 것
           계층 1: 물리 계층
                2: 데이터 링크 계층
                3: 네트워크 계층
                4: 전송 계층
                5: 세션 계층
                6: 표현 계층
                7: 응용 계층

	- HTML(Hyper Text Markup Language)
          Hyper Text:링크를 클릭 다른문서 사이트로 이동
          Markup:태그를 사용해서 문서를 구분
          >> 웹에서 자유롭게 오갈수 있는 웹문서를 만드는 언어  
        
	- HTTP(Hyper Text Transfer Protocol)
          WWW 상에서 정보를 주고받을 수 있는 프로토콜(규약)
		1. Head / Body(Multipart)로 구분
		   MIME(Multipurpose Internet Mail Extensions)-Type
                    전자 우편을 위한 인터넷 표준 포맷이다.
		2. Get(open) / Post(hidden) 방식
		3. Request(응답) / Response(요청) 
                응답해서 요청을 처리하면 => disconnect(접속 끊음)
		4. HTTP Status Code(ex 200 >> 성공)
                1xx (조건부 응답)
                2xx (성공)
                3xx (리다이렉션 완료) 표준 스트림의 흐름을 바꿈. 
                4xx (요청 오류)
                5xx (서버 오류)

	- CGI(Common Gateway Interface)
           웹 서버 상에서 사용자 프로그램을 동작시키기 위한 조합

	- HTTPS(Secure Socket Layer)
          월드 와이드 웹 통신 프로토콜인 HTTP의 보안이 강화된 버전
